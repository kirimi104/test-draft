【ポケモンユナイト】フィアレスドラフトツール 
--------------------------------------------------

このツールをダウンロードしていただきありがとうございます！
ドラフト（フィアレス形式）をスムーズに進めるためのツールです。

■ 内容物
1. 【ポケモンユナイト】フィアレスドラフトツール_ver06.html（本体）
2. img フォルダ（ポケモンの画像データ）
3. readme.txt（このファイル）

■ 使い方
1. zipファイルを右クリックし「すべて展開」または「解凍」してください。
2. 展開されたフォルダ内にある「【ポケモンユナイト】フィアレスドラフトツール_ver06.html」をダブルクリックしてください。
3. お使いのブラウザ（Google Chrome, Microsoft Edge等）でツールが起動します。

■ 注意事項
・「img」フォルダの名前を変えたり、削除したりしないでください。
（削除すると、ツール内でポケモンの画像が表示されなくなります）
・Googleドライブ上で直接開くとコードが表示されてしまうため、
　必ず一度「PCに保存（ダウンロード）」してから開いてください。

■ 推奨環境
・PCのブラウザ（Google Chrome / Edge / Brave）
・画面解像度 1280px 以上を推奨
